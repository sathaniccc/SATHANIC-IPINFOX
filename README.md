<p align="center">

## ✨ Features

- 🛰️ Get location info (City, Country, Coordinates)
- 🏢 Show ISP (Internet Service Provider)
- ⏰ Display Timezone
- 🔎 Fast lookup using public IP or domain
- ⚙️ Works on **Termux**, Linux, and macOS
